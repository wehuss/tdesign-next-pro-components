# Getting Started

TDesign Pro Components is a Vue 3 component library that provides enhanced components based on TDesign Vue Next. It aims to offer production-ready components with additional features and customizations.

## What is TDesign Pro Components?

TDesign Pro Components extends the capabilities of TDesign Vue Next by providing:

- **Enhanced Components**: Advanced features built on top of TDesign components
- **Production Ready**: Battle-tested components for enterprise applications
- **Type Safety**: Full TypeScript support with comprehensive type definitions
- **Modern Stack**: Built with Vue 3, Vite, and latest web technologies
