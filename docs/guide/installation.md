# Installation

## Using Package Manager

We recommend using a package manager like `pnpm`, `npm` or `yarn` to install TDesign Pro Components.

::: code-group

```bash [pnpm]
pnpm add tdesign-pro-components
```

```bash [npm]
npm install tdesign-pro-components
```

```bash [yarn]
yarn add tdesign-pro-components
```

:::

## CDN

You can also use TDesign Pro Components via CDN. This is useful for prototyping or when you want to avoid a build step.

```html
<!-- Vue 3 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<!-- TDesign Vue Next -->
<script src="https://unpkg.com/tdesign-vue-next/dist/tdesign.min.js"></script>
<!-- TDesign Pro Components -->
<script src="https://unpkg.com/tdesign-pro-components/dist/index.umd.js"></script>
```

## Peer Dependencies

TDesign Pro Components requires the following peer dependencies:

- `vue` >= 3.0.0
- `tdesign-vue-next` >= 1.0.0

Make sure to install them in your project:

::: code-group

```bash [pnpm]
pnpm add vue tdesign-vue-next tdesign-icons-vue-next
```

```bash [npm]
npm install vue tdesign-vue-next tdesign-icons-vue-next
```

```bash [yarn]
yarn add vue tdesign-vue-next tdesign-icons-vue-next
```

:::
